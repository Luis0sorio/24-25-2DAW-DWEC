# Ejercicio 12: Cambiar el contenido y color de tarjetas al cargar, al hacer clic, y al hacer scroll
Objetivo:

Cambiar el color de fondo de todas las tarjetas al cargar la página (DOMContentLoaded).
Modificar el texto de una tarjeta al hacer clic en ella (click).
Cambiar el color de texto cuando la tarjeta se enfoque (focus).
Volver al color original al perder el foco (blur).
Cambiar el tamaño de la tarjeta al hacer scroll en la página (scroll).
